
# Payment Simulator Core

A lightweight backend API for simulating real-world payment flows, including transaction states, failures, and webhook events.

##  Features

- Simulate payment transactions
- Multiple payment states (pending, success, failed)
- Fake card behavior for testing edge cases
- Webhook support for event notifications

## Purpose

This project is built to help developers test payment integrations without using real money or external services.

## Getting Started

```bash
git clone https://github.com/GordenArcher/payment-simulator.git
cd payment-simulator
````
